# DiracGrid

Welcome to the DIRAC organization on GitHub!

## About

DiracGrid is a comprehensive distributed computing framework for managing workloads, data, and resources across heterogeneous computing infrastructures. Our mission is to provide robust, scalable solutions for scientific computing and data-intensive applications.

## Key Projects

- **DIRAC** - DIRAC services & agents
- **diracx** - Next-generation DIRAC services & tasks with modern architecture

- **diracx-web** - Modern web interface for `diracx`
- **diracx-charts** - Kubernetes Helm charts for deployment
